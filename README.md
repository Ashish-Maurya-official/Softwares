# Softwares

## Installation

### Prerequisites
- Windows 7 or later
- .NET Framework 4.5+
- Administrator access

### Steps

1. **Download the installer**
   - Go to the [Releases](https://github.com/Ashish-Maurya-official/Softwares/releases) page
   - Download the latest `Recovator_Install_v*.*.*.exe` file

2. **Run the installer**
   - Double-click the downloaded `.exe` file
   - Accept the User Account Control (UAC) prompt if prompted

3. **Follow installation wizard**
   - Select your preferred installation directory
   - Click "Next" to proceed through the steps
   - Click "Install" to complete the installation

4. **Complete installation**
   - Wait for the installation to complete
   - Click "Finish" when done
   - Recovator will be added to your Start Menu and Desktop

5. **Launch the app**
   - Double-click the Recovator desktop shortcut, or
   - Search for "Recovator" in the Windows Start Menu
   - Click to launch the application
